# nijaru.github.io

Personal website and portfolio.

## Stack

- **Astro v5**
- **Tailwind CSS v4**
- **Bun**
- **LaTeX** (Resume generation)

## Development

```bash
bun install          # Install dependencies
bun run dev          # Start dev server
bun run build        # Build for production (includes resume PDF)
bun run build:resume # Generate resume PDF only
```

> **Note:** Resume generation requires XeLaTeX to be installed.
